# Areas

Rooms and zones of the house — one file per area (e.g. `kitchen.md`, `garage.md`,
`primary-bath.md`, `exterior.md`, `network-closet.md`).

An area file lists what's installed there (linking `[[item]]`s and `[[system]]`s),
plus area-specific notes (paint colors, dimensions, quirks). Add files as the house
gets mapped — no need to seed empties.
