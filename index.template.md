# Abode 101, Index

The navigation layer. An agent reads this first, then follows links to only what it
needs. Copy this to `index.md` and grow it as you capture. (See [`AGENTS.md`](AGENTS.md).)

## Map
- **[`areas/`](areas/)**: rooms and zones. One file per area.
- **[`systems/`](systems/)**: HVAC, electrical, plumbing, networking, lighting, safety, smart-home.
- **[`items/`](items/)**: things owned: devices, appliances, fixtures, tools.
- **[`maintenance/`](maintenance/)**: schedules and reminders.
- **[`references/`](references/)**: manuals, spec sheets, links, warranties.
- **[`resource_intake/`](resource_intake/)**: raw inbox; the ingest loop files it.
- **[`log.md`](log.md)**: the event stream (newest first).
